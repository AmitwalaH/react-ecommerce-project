## ğŸ”§ **Installation & Setup**

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/AmitwalaH/react-ecommerce-project.git
   ```

2. **Install Dependencies**:
   
   - For Frontend 
   ```bash
   cd ecommerce-project
   npm i
   ```
   - For Backend
     
   ```bash
   cd ecommerce-backend
   npm i
   ```

4. **Run the Project**:
   
    - For Frontend
      
   ```bash
   cd ecommerce-project
   npm run dev
   ```
   - For Backend
     
   ```bash
   cd ecommerce-backend
   npm run dev
   ```
